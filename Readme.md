# Introduction

this project aims to use state-of-the-art vision models: (a) open-vocabulary detection (b) open-vocabulary recognition (c) open vocbulary segmentation to perform automatic annotation to bootstrap the annotation task. 

We will try to use bunch of images stored in the folder and input specific keywords, the model can also allow custom changes like different image resize operation, SAHI-like slicing, etc. also different custom detection thresholds.
